# Public Dashboard Field Review

Review lens: public-facing pilot dashboard for Renewable Energy Long Island, prioritizing clarity, usefulness, sensitivity, and suitability for a general audience.

No dashboard code was changed as part of this review.

## Disposition Summary

- **Keep in main dashboard:** 28 items
- **Move to expandable detail section:** 41 items
- **Move to internal appendix:** 12 items
- **Remove from public view:** 3 items
